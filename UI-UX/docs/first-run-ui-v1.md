# Первый запуск UI V1

Инструкция для первого локального запуска интерфейса через Docker.

## 1. Установить инструменты

Нужны:

- Git for Windows;
- Docker Desktop for Windows.

После установки Git будет доступен Git Bash. Перед запуском проекта Docker Desktop должен быть открыт и полностью загружен.

## 2. Открыть терминал

Можно использовать:

- Git Bash;
- PowerShell;
- терминал в VS Code.

Ниже команды приведены для Git Bash. Их можно выделить, скопировать и вставить в терминал как обычный текст.

## 3. Перейти в папку, где будет храниться проект

Пример:

```bash
cd ~/Documents/GitHub
```

Путь может быть другим. Выберите папку, где хотите хранить проект локально.

## 4. Клонировать репозиторий

```bash
git clone https://github.com/NeuronsUII/PKB_neuroassistant.git
```

## 5. Перейти в папку проекта

```bash
cd PKB_neuroassistant
```

## 6. Переключиться на ветку с UI V1

```bash
git checkout feature/ui-import-frontend-v1
```

## 7. Перейти в папку первой версии интерфейса

```bash
cd UI-UX/frontend-v1
```

## 8. Запустить проект через Docker

```bash
docker compose up --build
```

При первом запуске Docker может скачивать базовые образы, поэтому сборка может занять несколько минут. Для этого шага нужен интернет.

## 9. Открыть интерфейс

В браузере открыть:

```text
http://localhost:3000
```

## 10. Чтобы остановить проект

В том же окне терминала нажать:

```text
Ctrl + C
```

Или выполнить:

```bash
docker compose down
```

## Если что-то не запустилось

- Проверьте, что Docker Desktop запущен.
- Проверьте, что команда выполняется из папки `UI-UX/frontend-v1`.
- Проверьте, что порт `3000` не занят другим приложением.
- Если команда `docker` не находится, перезапустите терминал после установки Docker Desktop.
