# QA Checklist

## Перед merge

- проект запускается локально
- проект собирается без ошибок
- основные вкладки открываются
- ссылки и кнопки работают
- layout не ломается на основном экране
- нет временных файлов в коммите

## Перед демонстрацией

- проверен запуск через Docker
- проверены ключевые сценарии
- актуальны тексты и подписи
- нет битой кириллицы
- есть понятный путь показа по вкладкам
