# План объединенного интерфейса UI V1

Ветка: `feature/ui-final`.

Цель: оставить UI V1 базовой версией и аккуратно перенести в нее сильные решения из альтернативных интерфейсов Елены и Гульнары без ломки основной логики.

## Что перенесено

1. Из версии Елены взят обзор базы знаний по разделам.
   Добавлено: `База знаний` -> `База знаний по разделам`.
   Сейчас карточки сделаны компактными, а разделы приближены к судостроительным темам: корпус, энергетика, электрика, трубопроводы, вентиляция, автоматика, навигация и связь, пожарная безопасность, спасательные средства, материалы, сварка, экология.

2. Из версии Елены взята идея работы внутри проектов.
   Добавлено: левая навигация -> `Чат` -> `Мои проекты` с вложенными чатами.

3. Из версии Елены взята идея поиска по разделам базы знаний.
   Добавлено: `Поиск` -> верхний блок `Поиск по базе знаний`.

4. Из версии Елены взяты действия для загрузки документов и загрузки по ссылке.
   Добавлено: `База знаний` -> верхний блок действий.

5. Из версии Елены взят контур управления контентом базы знаний.
   Добавлено: `Администрирование` -> `Управление контентом базы знаний`.

6. Из версии Гульнары взят стиль входа по ролям.
   Добавлено: экран входа -> поля `Логин`, `Пароль` и три карточки ролей: `Пользователь`, `Администратор знаний`, `Системный администратор`.

7. Из версии Гульнары взят проектный контекст в рабочей области.
   Добавлено: левая навигация -> дерево проектов и вложенных чатов под пунктом `Чат`.
   Убрано: лишние подписи этапа, системы и сессии.

8. Из версии Гульнары взята идея расширенного просмотра источника.
   Добавлено: `Чат` -> правое preview после клика на `Страница` или `Документ`.
   Сейчас в preview есть масштабирование, поиск по открытому источнику и крупный просмотр по кнопке раскрытия.

9. Из ответов заказчика и идеи истории диалогов добавлены срезы истории.
   Добавлено: `История` -> карточки `Диалоги`, `Проекты`, `Совпадения`, `Ответы с источниками`.

## Что убрано после проверки

1. Из `База знаний` убран отдельный блок `Очередь обработки документов`, чтобы не дублировать таблицу документов.
2. Из `Администрирование` убран блок `Что нужно от Gateway`.
3. Из `Администрирование` убран блок `CAD-метаданные для сверки`.
4. Из `Чат` убран блок `Фильтр разделов`.
5. Из `Чат` убрана постоянная правая панель `Контекстные документы`; справа остается только preview источников после клика.
6. Из верхней шапки убран выпадающий список проекта; выбор проекта перенесен в дерево чатов слева.

## Что нужно проверить глазами

1. `Вход`: есть логин, пароль, три карточки ролей; карточка подставляет нужный демо-логин и пароль.
2. `Рабочая область`: шапка по высоте совпадает с бренд-блоком; справа стоят пользователь, `Выйти` и демо-режим.
3. `Навигация`: под `Чат` раскрывается дерево `Мои проекты`; чаты можно выбрать, переименовать и удалить.
4. `Чат`: нет фильтра разделов и постоянной панели контекстных документов; preview открывается только по ссылкам `Страница` и `Документ`.
5. `Preview`: правая область отделена жирной вертикальной линией, есть масштаб, поиск по источнику и крупный центральный просмотр.
6. `Поиск`: виден блок разделов базы знаний; поиск и фильтры не разъехались.
7. `База знаний`: сверху кнопки загрузки, ниже 4 карточки состояния, затем разделы базы знаний, затем таблица документов.
8. `История`: видны срезы истории, фильтры, раскрытие диалога и продолжение чата.
9. `Администрирование`: нет блоков `Что нужно от Gateway` и `CAD-метаданные для сверки`.
10. Светлая тема выполнена в более голубой палитре версии Гульнары, темная тема не ломается.
11. Интерфейс собирается командой `npm run build`.
