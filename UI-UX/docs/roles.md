# Roles

## Разработчик

- правит UI и логику экранов
- проверяет локальный запуск
- описывает изменения в PR

## Тестировщик

- проверяет сценарии руками
- сверяет поведение с ТЗ
- фиксирует баги и замечания

## Коммуникатор

- собирает вопросы и обратную связь
- обновляет заметки по демонстрациям
- помогает не терять договоренности

## PM

- распределяет задачи
- следит за сроками и статусами
- определяет, что идет в демо-версию
