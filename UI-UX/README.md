# UI-UX Workspace

Общая зона команды UI/UX для проекта `PKB_neuroassistant`.

## Структура

- `frontend-v1/` — первая рабочая версия фронтенда
- `frontend-v2/` — альтернативная версия фронтенда
- `docs/` — правила работы, роли и QA-чеклисты

## Как работаем

1. Основная стабильная ветка проекта — `main`.
2. Общая рабочая ветка команды — `develop`.
3. Каждый участник создает свою ветку от `develop`.
4. Изменения заливаются через Pull Request в `develop`.
5. В `main` попадает только проверенная версия.

## Что важно

- Не работаем напрямую в `main`.
- Не коммитим `node_modules`, `dist`, временные файлы и скриншоты.
- Для первой версии UI используем папку `frontend-v1/`.
