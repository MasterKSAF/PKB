# UI Final

Объединенная версия интерфейса UI/UX для PKB Neuroassistant. Ветка предназначена для сборки итогового варианта интерфейса из текущей рабочей версии и удачных решений альтернативных интерфейсов.

Интерфейс работает в демонстрационном режиме без backend, но структура экранов уже подготовлена под дальнейшую стыковку с Gateway.

## Что находится в папке

| Путь | Что это |
| --- | --- |
| `frontend/` | Рабочее React-приложение объединенного интерфейса. Здесь исходный код, Dockerfile, nginx-конфиг, Vite, MUI и mock-данные. |
| `docs/first-run-ui-final.md` | Короткая инструкция первого запуска локально через Docker или npm. |
| `docs/ui-final-gateway-adaptation-plan-2026-05-27.md` | План адаптации UI Final к backend-слою Gateway. |
| `docs/ui-final-gateway-current-status-2026-06-03.md` | Текущий статус экспериментального подключения UI Final к актуальному Gateway. |
| `docs/ui-final-review-checklist-2026-05-27.docx` | Новый развернутый чек-лист проверки объединенной версии в формате Word. |

## Основные экраны

- `Вход`: логин, пароль и быстрый выбор демонстрационной роли.
- `Чат`: вопросы инженера, ответы ассистента, источники, предпросмотр страниц и документов, поиск по текущему чату, обратная связь.
- `Поиск`: поиск по базе знаний, фильтры, результаты и открытие источников.
- `База знаний`: состояние документов, разделы базы знаний, OCR и индексирование.
- `Проверка`: сверка проектного решения с требованиями НСИ, выбор документов, статусы и экспорт результата.
- `История`: поиск по диалогам, фильтры, раскрытие найденного диалога.
- `QA`: контрольные метрики, оценка ответов ассистента, журнал проверки.
- `Администрирование`: пользователи, роли, права, журналы и служебные настройки.

## Быстрый запуск

Если нужно забрать только эту версию, используйте ZIP ветки или sparse checkout из `docs/first-run-ui-final.md`.

```bash
git checkout feature/ui-final
cd "UI-UX/UI Final/frontend"
docker compose up --build
```

После запуска открыть:

```text
http://localhost:3000
```

Альтернативный запуск для разработки:

```bash
npm ci
npm run dev -- --host 127.0.0.1 --port 3310 --strictPort
```

Открыть:

```text
http://127.0.0.1:3310
```
