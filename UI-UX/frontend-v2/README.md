# Frontend V2

Папка под альтернативную версию интерфейса UI/UX.

Пока эта версия не является основной и не готова для запуска. Базовая рабочая версия находится в `../frontend-v1/`.

## Когда использовать

`frontend-v2` можно использовать для экспериментов с другим визуальным подходом, UI-библиотекой или структурой экранов.

Если участник команды делает самостоятельный вариант интерфейса, внутри этой папки должны появиться:

- исходный код проекта;
- `README.md` с описанием версии;
- инструкция запуска;
- Docker-файлы, если версия должна запускаться через Docker;
- краткое описание, чем эта версия отличается от `frontend-v1`.

## Важно

Не смешивать экспериментальную версию с `frontend-v1`, пока команда не решит, что изменения нужно переносить в базовую рабочую версию.
