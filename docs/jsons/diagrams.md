## 6. Сравнительная таблица ключевых полей между файлами

```
╔══════════════════════════════════════════════════════════════════════════╗
║                         Сравнение архитектур                            ║
║                                                                          ║
║  ┌──────────────────────────────────────────────────────────────────┐    ║
║  │  container_metadata                                              │    ║
║  │  schema_version, created_at, parser, source                      │    ║
║  └──────────────────────────────────────────────────────────────────┘    ║
║                            │                                              ║
║                            ▼                                              ║
║  ┌──────────────────────────────────────────────────────────────────┐    ║
║  │  document_metadata                                               │    ║
║  │  gost_code, full_title, normalized_title,                        │    ║
║  │  adoption (объект), amendments,                                  │    ║
║  │  validity_restriction_removed (объект)                           │    ║
║  └──────────────────────────────────────────────────────────────────┘    ║
║                            │                                              ║
║                            ▼                                              ║
║  ┌──────────────────────────────────────────────────────────────────┐    ║
║  │  content                                                         │    ║
║  │  text → nsi_document_sections                                    │    ║
║  │  tables → nsi_document_sections + columns                        │    ║
║  │  figures → nsi_document_sections                                 │    ║
║  │  formulas → nsi_document_sections                                │    ║
║  └──────────────────────────────────────────────────────────────────┘    ║
║                            │                                              ║
║                            ▼                                              ║
║  ┌──────────────────────────────────────────────────────────────────┐    ║
║  │  cross_references                                                │    ║
║  │  → nsi_cross_references                                          │    ║
║  │  Добавлено: is_resolved, resolved_document_id,                   │    ║
║  │  created_at, source_document_id                                  │    ║
║  └──────────────────────────────────────────────────────────────────┘    ║
╚══════════════════════════════════════════════════════════════════════════╝
```

---

> **Условные обозначения:** `•` — поле/свойство, `→` — трансформация/маппинг, `массив` — array, `объект` — object.
