API Нейроассистента ПКБ (v1.0)
===============================

### Общие положения

- Базовый URL (оркестратор): ```https://{host}/api/v1```
 
- Базовый URL для внутренних запросов: ```https://127.0.0.1/api/v1```

- Формат данных: ```application/json```, для загрузки файлов – ```multipart/form-data```

- Аутентификация: все запросы, кроме `````/auth/* и /health`````, требуют заголовок  
  ```Authorization: Bearer <access_token>```. Токен получается через ```/auth/token```.

- Формат ошибок (унифицирован):

- json

``` json
{

  "error": {

    "code": 422,
    
    "code_name": "INVALID_FORMAT",

    "message": "Поддерживаются только PDF и изображения",

    "details": {}

  }

}
```

<a href="https://developer.mozilla.org/en-US/docs/Web/HTTP/Reference/Status">Коды ответов HTTP</a>

Коды ответа:

| code | code name             | message                                           | Коментарий                                                                |
|------|-----------------------|---------------------------------------------------|---------------------------------------------------------------------------|
| 200  | OK                    | Успех                                             |                                                                           |
| 201  | CREATED               | Создан ресурс                                     |                                                                           |
| 202  | ACCEPTED              | Запрос принят                                     | Когда нет немедленного ответа, например, поставить документ на обработку. |
| 400  | BAD_REQUEST           | Неверные параметры                                |                                                                           |
| 401  | UNAUTHORIZED          | Нет доступа - клиент не известен                  |                                                                           |
| 403  | FORBIDDEN             | Нет доступа - нет прав на ресурс, клтент известен |                                                                           |
| 404  | NOT_FOUND             | Не найдено, нет такого адреса                     |                                                                           |
| 404  | NOT_FOUND             | Не найдено, нет ресурса                           |                                                                           |
| 422  | UNPROCESSABLE_FORMAT  | Ошибка валидации/ семантическая ошибка            |                                                                           |
| 500  | INTERNAL_SERVER_ERROR | Внутренняя ошибка сервера                         |                                                                           |
| 501  | NOT_IMPLEMENTED       | Метод не внедрён                                  | GET and HEAD не должны возвращать этот код                                |
